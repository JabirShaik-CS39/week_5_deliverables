Tailwind CSS & Shadcn/UI – Complete Study Notes
1. Introduction to Tailwind CSS
What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework that provides small reusable utility classes to build modern user interfaces directly in HTML/JSX without writing custom CSS for every component.

Unlike traditional CSS frameworks such as Bootstrap, Tailwind does not provide pre-built components. Instead, it provides utility classes that can be combined to create custom designs.

Example

Traditional CSS:

<button class="btn">Login</button>
.btn {
  background: blue;
  color: white;
  padding: 10px 20px;
  border-radius: 8px;
}

Tailwind CSS:

<button class="bg-blue-500 text-white px-5 py-2 rounded-lg">
  Login
</button>

No separate CSS file is required.

2. Utility-First Philosophy
What is Utility-First?

A utility class performs one specific job.

Examples:

text-center
bg-red-500
p-4
mt-5
rounded-lg
flex
grid

Each class controls one CSS property.

Example
<div class="bg-white p-6 rounded-lg shadow-lg">
  Product Card
</div>

Equivalent CSS:

background-color: white;
padding: 24px;
border-radius: 12px;
box-shadow: ...
Advantages
Faster Development

No need to switch between CSS and HTML.

Consistent Design

Uses predefined spacing, colors, typography.

Responsive Design

Built-in responsive utilities.

Smaller CSS Bundle

Unused classes are removed during production build.

Easy Maintenance

No complex CSS architecture.

3. Responsive Design in Tailwind

Tailwind uses mobile-first responsive design.

Default styles apply to mobile.

Larger screens override them.

Breakpoints
Prefix	Minimum Width
sm	640px
md	768px
lg	1024px
xl	1280px
2xl	1536px
Example
<div class="text-sm md:text-lg lg:text-2xl">
  Responsive Text
</div>

Meaning:

Mobile → small text
Tablet → large text
Desktop → extra-large text
Responsive Width
<div class="w-full md:w-1/2 lg:w-1/3">
  Content
</div>
Responsive Padding
<div class="p-2 md:p-4 lg:p-8">
  Box
</div>

5. Grid with Tailwind

Grid is used for two-dimensional layouts.

Basic Grid
<div class="grid grid-cols-3 gap-4">

Creates 3 columns.

Responsive Grid
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-6">

Result:

Mobile → 1 column
Tablet → 2 columns
Desktop → 4 columns

Perfect for e-commerce product pages.

6. Dark Mode in Tailwind

Dark mode allows users to switch between light and dark themes.

Class Strategy

In tailwind.config.js:

module.exports = {
  darkMode: "class",
}
Usage
<div class="bg-white dark:bg-gray-900">

Meaning:

Light mode:

background: white;

Dark mode:

background: gray-900;
Text Example
<p class="text-black dark:text-white">
  Hello
</p>
Toggle Dark Mode
document.documentElement.classList.toggle("dark");

8. Introduction to Shadcn/UI
What is Shadcn/UI?

Shadcn/UI is a modern component library built using:

React
Tailwind CSS

Unlike traditional libraries:

Full customization.

Benefits

Customizable

Own the source code.

Beautiful Design

Production-ready components.

Tailwind Integration
